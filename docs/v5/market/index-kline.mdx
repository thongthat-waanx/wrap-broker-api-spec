Query for historical <a href="https://waanx.com/en-US/help-center/s/article/Glossary-Waanx-Trading-Terms">index price</a> klines. Charts are returned in groups based on the requested interval.
> **Covers: USDT contract / USDC contract / Inverse contract**

### HTTP Request
<APIEndpoint method="GET" url="/v5/market/index-price-kline" />

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------|
|[category](../enum#category) |false |string |Product type. `linear`,`inverse` <ul><li>When `category` is not passed, use `linear` by default</li></ul>|
|symbol |**true** |string |Symbol name, like `BTCUSDT`, uppercase only|
|[interval](../enum#interval) |**true** |string |Kline interval. `1`,`3`,`5`,`15`,`30`,`60`,`120`,`240`,`360`,`720`,`D`,`W`,`M`|
|start |false |integer |The start timestamp (ms)|
|end |false |integer |The end timestamp (ms)|
|limit |false |integer |Limit for data size per page. [`1`, `1000`]. Default: `200`|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Product type |
|symbol |string |Symbol name |
|list |array |<ul><li>An string array of individual candle</li><li>Sort in reverse by `startTime`</li></ul>|
|> list[0]: startTime |string |Start time of the candle (ms) |
|> list[1]: openPrice |string |Open price |
|> list[2]: highPrice |string |Highest price |
|> list[3]: lowPrice |string |Lowest price |
|> list[4]: closePrice |string |Close price. _Is the last traded price when the candle is not closed_|
