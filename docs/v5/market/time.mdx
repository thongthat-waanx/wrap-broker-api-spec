### HTTP Request
<APIEndpoint method="GET" url="/v5/market/time" />

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| timeSecond | string | Waanx server timestamp (sec)  |
| timeNano | string | Waanx server timestamp (nano)  |
