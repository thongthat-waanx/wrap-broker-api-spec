獲取Waanx服務器時間

:::info
* 在極端市場波動期間, 此介面可能會出現延遲增加或資料傳遞暫時延遲的情況
:::

### HTTP 請求
<APIEndpoint method="GET" url="/v5/market/time" />

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| timeSecond | string | Waanx服務器時間戳 (秒)  |
| timeNano | string | Waanx 服務器時間戳 (微秒)  |

<a href="/api-explorer/v5/market/time"><Button>運行實例 >></Button></a>

### 請求示例
<Tabs groupId="programming-languages">
<TabItem value="http" label="HTTP">

```http
GET /v5/market/time HTTP/1.1
Host: api.waanx.com
```

</TabItem>
<TabItem value="python" label="Python">

```python
from pybit.unified_trading import HTTP
session = HTTP(testnet=True)
print(session.get_server_time())
```

</TabItem>
<TabItem value="Go" label="Go">

```go
import (
	"context"
	"fmt"
	waanx "github.com/waanx-exchange/waanx.com"
)
client := waanx.com("", "", waanx.com(waanx.com))
client.NewUtaWaanxServiceNoParams().GetServerTime(context.Background())
```

</TabItem>
<TabItem value="java" label="Java">

```java
import com.waanx.com;
var client = waanx.com().newAsyncMarketDataRestClient();
client.getServerTime(System.out::println);
```

</TabItem>
<TabItem value="Node" label="Node.js">

```n4js
const { RestClientV5 } = require('waanx-api');

const client = new RestClientV5({
  testnet: true,
});

client
  .getServerTime()
  .then((response) => {
    console.log(response);
  })
  .catch((error) => {
    console.error(error);
  });
```

</TabItem>
</Tabs>
